#

## Install Memo

### TypeScript

```
npm install -D typescript @types/node ts-node
```

### Packages

```
npm install express
```

```
npm install --save-dev @types/express
```

### nodemon

```
npm install --save-dev nodemon
```

`nodemon.json`

```
{
    "watch": [
        "src"
    ],
    "ext": "ts",
    "exec": "ts-node ./src/index.ts"
}
```

### TypeScript Initialize

```
npx tsc --init
```
